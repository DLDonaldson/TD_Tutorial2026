# TD_Tutorial2026
This github repository contains the information associated with the IEEE Transmission and Distribution Tutorial 2026 on Reliability and Resilience. Further materials will be made following delivery of the tutorial.

## Outline:

- **Introductions**

- **Part 1**: Introduction to IEEE 1366 and 1782 Standards

- **Part 2**: Implementing methods to calculate reliability

- **Lunch**

- **Part 3**: Introduction to the IEEE Distribution Resiliency Guide

- **Part 4**:Implementing methods to calculate resilience

- **Part 5**: Strategies for adaptation and mitigation
